# 我的餐廳清單

運用node.js與express.js打造屬於自己的餐廳清單

## 安裝
---

1. 下載 restaurant_list資料夾

    ```
    git clone https://github.com/austin72905/restaurant-list.git
    ```

2. 開啟終端機，進入此資料夾

    ```
    cd restaurant_list
    ```

3. 安裝npm套件

    ```
    npm install
    ```

4. 執行專案

    ```
    npm run dev
    ```

5. 在瀏覽器中輸入網址 http://localhost:3000

## 特色
---
1. 能於搜尋欄透過關鍵字找到餐廳
2. 點擊餐廳能取得詳細資料，如地址、電話
3. 餐廳資料此用json檔案匯入，方便管理
